# 🎓 Inspire Academy - Best Institute for Intermediate Students

A modern, responsive website for Inspire Academy, the best institute for intermediate students in Andhra Pradesh. Built with React.js, featuring mobile-first design, SEO optimization, and comprehensive coaching programs for Class 11 & 12 students.

## 🌟 Features

### 📱 **Mobile-First Responsive Design**
- Fully responsive across all devices (mobile, tablet, desktop)
- Touch-friendly navigation and interactions
- Optimized performance for mobile networks
- Progressive Web App capabilities

### 🎯 **SEO Optimized**
- Best keywords for "intermediate students" and "Class 11 & 12 coaching"
- Meta tags optimized for search engines
- Fast loading and mobile-friendly
- Structured data for better rankings

### 📚 **Comprehensive Pages**
- **Home**: Hero section with animated statistics
- **About**: Institute information and achievements
- **Courses**: NEET, JEE, EAMCET coaching programs
- **Admissions**: Registration and enrollment process
- **Results**: Success stories and testimonials
- **Blog**: Study tips and educational content
- **FAQ**: Common questions and answers
- **Contact**: Location, phone, WhatsApp integration

- **WhatsApp Integration**: Direct messaging for inquiries

## 📦 **Installation & Setup**

### Prerequisites
- Node.js 16+ 
- npm or yarn
- Git

### Local Development
```bash
# Clone the repository
git clone https://github.com/konapalask/inspie_academy.git

# Navigate to project directory
cd inspie_academy

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

### Environment Setup
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:5173
```

## 🔧 **Configuration**

### Vercel Deployment
```json
{
  "rewrites": [
    {
      "source": "/(.*)",
      "destination": "/index.html"
    }
  ],
  "framework": "vite"
}
```

```javascript
module.exports = {
  content: ["./src/**/*.{js,jsx,ts,tsx}"],
```
